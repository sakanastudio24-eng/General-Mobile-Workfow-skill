## Suggested Starter Paths

1. Cross-platform speed:
   - Start with Expo managed workflow and EAS Build/Update.
2. Heavy native SDK integration:
   - Start with Expo prebuild or bare React Native with explicit native modules.
3. Fully native product:
   - Use Swift for iOS and Kotlin for Android with shared backend contracts.
4. Shared business logic with native UI:
   - Use Kotlin Multiplatform for domain/network modules and keep UI platform-native.
